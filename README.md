# soc-home-lab
SOC-focused home lab using Windows 10 and Kali Linux to simulate, investigate and document security events including authentication activity, network scanning and endpoint telemetry.


Overview

This repository documents my SOC-focused home lab built using Windows 10 and Kali Linux virtual machines. The purpose of this lab is to simulate real-world security events, investigate system and network activity, and develop practical skills aligned with Security Operations Centre (SOC) analyst roles.

Environment
- Windows 10 VM (Event Viewer, Sysmon, Security Logs)
- Kali Linux VM (Network scanning and activity generation)
- VMware Workstation

Objectives
- Investigate Windows authentication events
- Analyse network scanning and reconnaissance activity
- Understand endpoint telemetry and system logs
- Develop incident investigation and reporting skills
- Build SOC-relevant documentation for portfolio and CV

Skills Being Developed
- Windows Event Log analysis
- Network traffic awareness
- Security incident investigation
- Log correlation and analysis mindset
- Basic detection and monitoring techniques
